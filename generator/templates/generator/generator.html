Getting Started

  1. Move the playhead to where a subtitle should begin.
  2. Click Add Cue! to create a new subtitle.
  3. Write the subtitle text in the editor.
  4. Adjust the start and end times if needed.
  5. Click Save! to store your changes.
  6. Select any cue in the sidebar to edit it later.
  7. Click the × button to remove a subtitle.
  8. When finished, click Create VTT File!.

Tip: New cues are automatically created at the video's current position and last 5 seconds by default.
